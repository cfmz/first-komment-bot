# ⚡ Telegram Comment Bot

Автоматический комментатор для Telegram-каналов.

### Возможности
- 🔐 Вход через аккаунт Telegram
- ▶️ Мгновенный комментарий под новыми постами
- ✏️ Настраиваемый текст (можно {time} и {date})
- 📋 Управление каналами (добавить/удалить)
- 🌍 Режим «Все каналы»
- 🎨 Цветные Inline-кнопки
- 🚀 Работает 24/7 на бесплатном хостинге

### Технологии
- Python + Telethon
- python-telegram-bot
- aiohttp (для веб-сервера)
- Render (Web Service)

### Установка
1. Получить API_ID и API_HASH на my.telegram.org
2. Получить BOT_TOKEN у @BotFather
3. Добавить переменные окружения:
   - API_ID
   - API_HASH
   - BOT_TOKEN
4. Запустить через Render

### Команды
- /start — главное меню
- /cancel — отмена текущего действия
